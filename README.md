# flight_booker
An Android application for booking airline tickets.
The application provides the functionality to store, retrieve, and update flight information, as well as information about our clients. A user is able to create a trip itinerary 1 by providing origin, destination, and travel dates information. The application calculates and displays itinerary costs and create new bookings. A user is able to display the list of suggested itineraries by the itinerary cost and by total travel time.
